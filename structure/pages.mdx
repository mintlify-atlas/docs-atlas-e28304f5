## Common structure

All pages share these common elements:

### Header navigation

```html
<header class="site-header">
  <div class="container nav">
    <a class="brand" href="../index.html">BuildPro Construction</a>
    <nav class="nav-links">
      <a href="../index.html">Home</a>
      <a href="about.html" class="active">About</a>
      <a href="services.html">Services</a>
      <a href="projects.html">Projects</a>
      <a href="contact.html">Contact</a>
    </nav>
  </div>
</header>
```

The `active` class is applied to the current page link.

### Footer

```html
<footer class="site-footer">
  <div class="container">© 2026 BuildPro Construction. All rights reserved.</div>
</footer>
```
