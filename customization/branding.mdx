## Customize brand colors

The website uses CSS custom properties (variables) for consistent theming. All colors are defined in `assets/css/base.css:1-10`.

### Color variables

```css
:root {
  --primary: #1e3a8a;      /* Primary blue - headers, buttons */
  --secondary: #f97316;     /* Secondary orange - accents */
  --bg: #f3f4f6;           /* Background gray */
  --text: #111827;         /* Main text color */
  --white: #ffffff;        /* White backgrounds */
  --muted: #6b7280;        /* Muted text */
  --radius: 14px;          /* Border radius */
  --shadow: 0 10px 25px rgba(17, 24, 39, 0.08); /* Box shadow */
}
```

<Steps>
  <Step title="Choose your brand colors">
    Identify your company's brand colors:
    - Primary color (for headers, primary buttons, key elements)
    - Secondary color (for accents, hover states, CTAs)
    - Background color (optional, light neutral tone)
  </Step>

  <Step title="Update CSS variables">
    Open `assets/css/base.css` and modify the `:root` section:

    ```css
    :root {
      --primary: #0066cc;      /* Your primary brand color */
      --secondary: #ff9900;    /* Your secondary/accent color */
      --bg: #f3f4f6;          /* Keep or customize background */
      --text: #111827;        /* Keep or customize text color */
      --white: #ffffff;
      --muted: #6b7280;
      --radius: 14px;         /* Adjust border roundness if desired */
      --shadow: 0 10px 25px rgba(17, 24, 39, 0.08);
    }
    ```
  </Step>

  <Step title="Test the changes">
    Open your website in a browser and verify:
    - Primary buttons use your primary color
    - Navigation links and headings reflect your brand
    - Hover states and accents use your secondary color
    - Overall appearance matches your brand identity
  </Step>
</Steps>

<Tip>
  Use tools like [Adobe Color](https://color.adobe.com) or [Coolors](https://coolors.co) to generate professional color palettes that work well together.
</Tip>

### Contact form endpoint

Update the form submission endpoint to receive inquiries:

**Locations:**
- `index.html:147` - Homepage contact form
- `pages/contact.html:32` - Contact page form

```html
<!-- Before -->
<form class="card" action="https://formspree.io/f/your-form-id" method="POST">

<!-- After -->
<form class="card" action="https://formspree.io/f/YOUR_ACTUAL_FORM_ID" method="POST">
```

<Note>
  Sign up at [Formspree.io](https://formspree.io) to get a free form endpoint. Replace `YOUR_ACTUAL_FORM_ID` with your unique form ID to start receiving submissions.
</Note>

### Google Maps location

Update the embedded map on `pages/contact.html:74` to show your office location:

```html
<!-- Before -->
src="https://maps.google.com/maps?q=Phnom%20Penh&t=&z=13&ie=UTF8&iwloc=&output=embed"

<!-- After -->
src="https://maps.google.com/maps?q=Your+City+Address&t=&z=13&ie=UTF8&iwloc=&output=embed"
```

<Tip>
  For a more precise map, visit [Google Maps](https://maps.google.com), search for your address, click "Share" → "Embed a map", and copy the iframe `src` URL.
</Tip>
