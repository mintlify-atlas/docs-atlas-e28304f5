## Get started in 2 minutes

BuildPro Construction is a static website that requires no build process or dependencies. You can run it immediately with your browser or a local development server.

<Steps>
  <Step title="Clone or download the repository">
    Get the source code on your local machine:

    ```bash
    git clone <your-repo-url>
    cd build-pro-construction
    ```

    Or download and extract the ZIP file from your repository.
  </Step>

  <Step title="Choose your preferred method to run locally">
    Since this is a static site with no build process, you have two simple options:

    ### Option 1: Open directly in browser

    Navigate to the project folder and double-click `index.html` to open it in your default browser.

    <Warning>
    Some features like external resources may not load correctly due to CORS restrictions when opening files directly. For the best experience, use Option 2.
    </Warning>

    ### Option 2: Use VS Code Live Server (recommended)

    If you're using Visual Studio Code:

    1. Install the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
    2. Open the project folder in VS Code
    3. Right-click `index.html` and select **Open with Live Server**
    4. Your browser will automatically open to `http://127.0.0.1:5500`

    <Info>
    Live Server provides automatic browser refresh when you edit files, making development much faster.
    </Info>

    ### Option 3: Use Python's built-in server

    If you have Python installed:

    <CodeGroup>
    ```bash Python 3
    python -m http.server 8000
    ```

    ```bash Python 2
    python -m SimpleHTTPServer 8000
    ```
    </CodeGroup>

    Then open `http://localhost:8000` in your browser.

    ### Option 4: Use Node.js http-server

    If you have Node.js installed:

    ```bash
    npx http-server -p 8000
    ```

    Then open `http://localhost:8000` in your browser.
  </Step>

  <Step title="Verify the site is running">
    You should see the BuildPro Construction homepage with:

    - Navigation header with links to Home, About, Services, Projects, Contact
    - Hero section with "We Build Your Future" heading
    - About section with company information
    - Services grid showing 6 service cards
    - Projects showcase gallery
    - Contact form and information
    - Footer

    <Note>
    Navigate to other pages using the header navigation to explore the full site.
    </Note>
  </Step>

  <Step title="Test the interactive features">
    Navigate to the **Projects** page (`pages/projects.html`) and test the filter functionality:

    - Click **All** to show all projects
    - Click **Residential** to filter residential projects only
    - Click **Commercial** to filter commercial projects only

    The active filter button should be highlighted, and projects should show/hide accordingly.
  </Step>
</Steps>

## Project structure overview

Once you have the site running, familiarize yourself with the project structure:

```
build-pro-construction/
├── index.html              # Homepage - start here
├── pages/                  # Additional pages
│   ├── about.html
│   ├── services.html
│   ├── projects.html       # Has interactive filter
│   └── contact.html
└── assets/
    ├── css/                # Modular stylesheets
    │   ├── main.css        # Import entrypoint
    │   ├── base.css        # Variables and reset
    │   ├── layout.css      # Header, footer, grid
    │   ├── components.css  # Buttons, cards
    │   ├── sections.css    # Page sections
    │   └── responsive.css  # Mobile styles
    └── js/
        └── projects-filter.js  # Filter logic
```

## Customization guide

Now that the site is running, you can customize it for your needs:

### Update brand information

Search and replace across all HTML files:

<CodeGroup>
```text Company name
Find: BuildPro Construction
Replace: Your Company Name
```

```text Contact phone
Find: +855 12 345 678
Replace: Your phone number
```

```text Email address
Find: info@buildpro.com
Replace: your@email.com
```

```text Location
Find: Phnom Penh, Cambodia
Replace: Your city, country
```
</CodeGroup>

### Configure the contact form

The contact form uses [Formspree](https://formspree.io) for form handling. Update the form action in `index.html` and `pages/contact.html`:

```html index.html (line 147)
<form class="card" action="https://formspree.io/f/your-form-id" method="POST">
  <!-- Update 'your-form-id' with your actual Formspree form ID -->
</form>
```

<Steps>
  <Step title="Create a Formspree account">
    Sign up at [formspree.io](https://formspree.io)
  </Step>
  <Step title="Create a new form">
    Create a form and copy your form endpoint URL
  </Step>
  <Step title="Update the action attribute">
    Replace `https://formspree.io/f/your-form-id` with your actual endpoint
  </Step>
</Steps>

### Customize theme colors

Edit the CSS custom properties in `assets/css/base.css`:

```css assets/css/base.css (lines 1-10)
:root {
  --primary: #1e3a8a;      /* Main brand color (blue) */
  --secondary: #f97316;    /* Accent color (orange) */
  --bg: #f3f4f6;           /* Background color */
  --text: #111827;         /* Text color */
  --white: #ffffff;        /* White */
  --muted: #6b7280;        /* Muted text */
  --radius: 14px;          /* Border radius */
  --shadow: 0 10px 25px rgba(17, 24, 39, 0.08);  /* Box shadow */
}
```

<Info>
Changing these variables will update colors across the entire website automatically.
</Info>

### Replace placeholder images

The demo uses Unsplash images. Replace them with your own:

1. Add your images to a new `assets/images/` folder
2. Update image `src` attributes in the HTML files
3. Use descriptive `alt` text for accessibility

```html Before
<img src="https://images.unsplash.com/photo-..." alt="Modern Villa" />
```

```html After
<img src="../assets/images/villa-project.jpg" alt="Modern Villa in Phnom Penh" />
```
