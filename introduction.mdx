## Welcome to BuildPro Construction

BuildPro Construction is a professional, production-ready static website designed for construction companies. Built with clean, semantic HTML and modular CSS architecture, this project demonstrates best practices for modern web development without the complexity of frameworks.

<Note>
This is a fully static website - no build process, no dependencies, and no server-side code required. Just open and run.
</Note>

## Project architecture

The project follows a clean, intuitive folder structure:

```
build-pro-construction/
├── index.html              # Homepage
├── README.md               # Project documentation
├── assets/
│   ├── css/
│   │   ├── base.css        # Variables, reset, typography
│   │   ├── layout.css      # Header, nav, footer, grid
│   │   ├── components.css  # Buttons, cards, reusable blocks
│   │   ├── sections.css    # Hero, gallery, page-specific UI
│   │   ├── responsive.css  # Breakpoints and mobile styles
│   │   └── main.css        # Single import entrypoint
│   └── js/
│       └── projects-filter.js  # Filter logic for projects
└── pages/
    ├── about.html          # Company information
    ├── services.html       # Service offerings
    ├── projects.html       # Project showcase with filters
    └── contact.html        # Contact form and information
```

## Key features

### Modular CSS system

The CSS is organized into focused modules that are imported through a single entrypoint:

```css assets/css/main.css
@import url("./base.css");
@import url("./layout.css");
@import url("./components.css");
@import url("./sections.css");
@import url("./responsive.css");
```

This approach provides:
- Clear separation of concerns
- Easy maintenance and updates
- Reusable component styles
- Scalable architecture

### CSS custom properties

The design system uses CSS variables for consistent theming:

```css assets/css/base.css
:root {
  --primary: #1e3a8a;
  --secondary: #f97316;
  --bg: #f3f4f6;
  --text: #111827;
  --white: #ffffff;
  --muted: #6b7280;
  --radius: 14px;
  --shadow: 0 10px 25px rgba(17, 24, 39, 0.08);
}
```

<Info>
Change colors across the entire site by updating a single variable in `base.css`
</Info>

### Vanilla JavaScript filtering

The projects page includes an interactive filter system built with vanilla JavaScript:

```javascript assets/js/projects-filter.js
document.addEventListener("DOMContentLoaded", () => {
  const filterButtons = document.querySelectorAll(".filter-btn");
  const projectCards = document.querySelectorAll(".project-item");

  filterButtons.forEach((button) => {
    button.addEventListener("click", () => {
      const selectedType = button.dataset.filter;

      filterButtons.forEach((item) => item.classList.remove("active"));
      button.classList.add("active");

      projectCards.forEach((card) => {
        const type = card.dataset.type;
        const shouldShow = selectedType === "all" || selectedType === type;
        card.style.display = shouldShow ? "block" : "none";
      });
    });
  });
});
```
