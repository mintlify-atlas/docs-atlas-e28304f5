## Container

The container centers content with responsive width constraints.

```css
.container {
  width: min(1120px, 92%);
  margin: 0 auto;
}
```

Maximum width is 1120px on large screens, reducing to 92% of viewport width on smaller screens for proper margins.

## Section

Sections provide consistent vertical spacing between page segments.

```css
.section {
  padding: 4.5rem 0;
}

.section-title {
  font-size: clamp(1.7rem, 1.2rem + 1.5vw, 2.3rem);
  line-height: 1.2;
  margin: 0 0 0.75rem;
}

.section-subtitle {
  margin: 0;
  color: var(--muted);
}
```

Sections have 4.5rem top and bottom padding. Section titles use fluid typography that scales between 1.7rem and 2.3rem based on viewport width.

## Grid systems

BuildPro uses CSS Grid for flexible layouts.

<CodeGroup>

```css Two-column grid
.grid-2 {
  display: grid;
  grid-template-columns: repeat(2, minmax(0, 1fr));
  gap: 2rem;
  align-items: center;
}
```

```css Three-column card grid
.card-grid {
  display: grid;
  grid-template-columns: repeat(3, minmax(0, 1fr));
  gap: 1.2rem;
}
```

```css Contact grid
.contact-grid {
  display: grid;
  grid-template-columns: 1.15fr 0.85fr;
  gap: 1.2rem;
}
```

</CodeGroup>

**Grid-2** creates two equal-width columns with 2rem gap and vertically centered items. Used for image-text layouts.

**Card-grid** displays three equal columns with 1.2rem spacing. Perfect for service cards and project showcases.

**Contact-grid** uses asymmetric columns (1.15fr and 0.85fr) to give the form more space than the contact info sidebar.
